# 短视频三段式标签字典 (v2.0 心理学重构版)
更新时间：2026-06-01

## 1. Hook 段（留人钩子）
### 1.1 命名与考量（底层理论）
基于“注意力心理学”的四大驱动链路，按照用户大脑处理信息的本能优先级进行排序重构：
1. **趋利避害链路（H1-H2）**：诉诸生存本能，瞬间触发警报（痛点恐惧）或占便宜心理（价格贪婪）。
2. **视觉感官链路（H3-H4）**：诉诸直觉，跳过大脑思考，直接用冲击力或爽感画面强控眼球。
3. **认知悬念链路（H5）**：诉诸好奇心，制造信息差与预期违背，强制大脑停留寻找答案。
4. **社交身份链路（H6-H7）**：诉诸归属感，通过身份共鸣或信任转移留住特定圈层。

### 1.2 定义与核心边界
*   **H1 痛点/焦虑直击**
    *   **定义**：诉诸避害。直接暴露症状、制造麻烦与焦虑（如卡粉特写、直击脱发）。
    *   **核心边界（排他）**：必须是纯粹的病症展示且无背景铺垫。若存在生活剧情或明确的时空/身份限定，绝对优先打 H6；无视觉反差蜕变，不打 H3。
*   **H2 利益/价格前置**
    *   **定义**：诉诸趋利。直接抛出具体的交易数字、价格落差或促销福利开场（不论是品牌高客单的买赠促销，还是白牌低价的绝对破价，只要起手诉诸算账与让利逻辑即属此类）。
    *   **核心边界（排他）**：前3-5秒必须出现明确的价格数字或福利词汇。无算账元素、仅凭画面强刺激绝对不能打 H2。
*   **H3 反差结果前置**
    *   **定义**：诉诸视觉终效冲击。极具视觉落差的 Before & After 蜕变。
    *   **核心边界（排他）**：必须是产品作用后真实的视觉反差。纯口播描述的成功结果、受众的情绪反应（如惊讶）绝不能判定为 H3。
*   **H4 即时操作展示**
    *   **定义**：诉诸视觉过程冲击。动作即看点，开局直接一镜到底顺滑操作，无情绪铺垫。
    *   **核心边界（排他）**：只要叠加了强痛点、强场景、强反差、强人设，H4 必须降级让位给其他强标签。
*   **H5 反常识与悬念**
    *   **定义**：诉诸认知冲突与信息差。拆分为三个二级子类：
        *   *H5-1 行为猎奇/反常规*：视觉与操作违背常理（如开水煮馒头、火鸡面+螺蛳粉）。
        *   *H5-2 认知/观点冲突*：直接挑战大众固有认知（如“全网夸的其实是智商税”）。
        *   *H5-3 质疑打假/亲测验证*：博主化身嘴替，对夸大宣传或负面评论提出质疑并实测。
    *   **核心边界（排他）**：核心驱动力必须是“违背常理、引发好奇、制造争议”。若落入纯痛点、纯视觉对比或纯算账逻辑，则不属于此类。
*   **H6 场景/人群代入**
    *   **定义**：诉诸社交归属。为防止外延过于宽泛，明确拆分为四个基础子标签：
        *   *H6-1 剧情/关系演绎*：用生活短剧、POV视角或关系冲突（夫妻/代际/职场）起手。
        *   *H6-2 节点/时空限定*：利用特定时间（如换季、节日、早八）或空间（如高铁、宿舍）制造场景规律性刚需。
        *   *H6-3 人群/特征点名*：直接呼唤具有特定社会身份（宝妈、手残党）或生理特征（黄黑皮、敏感肌）的受众。
        *   *H6-4 热点/公共话题代入*：利用突发不可预见的社会新闻、娱乐八卦、全网爆梗起手，强行建立“吃瓜/围观”的社交共识，借势引流（区别于H6-2的个人场景刚需，也区别于H5-3针对产品本身的打假）。
    *   **核心边界（排他）**：只要存在生活情境剧、明确的时令限定、特征点名或外部热点借势，即使包含痛点，也绝对优先打 H6，而非 H1。
*   **H7 明星/权威同款**
    *   **定义**：诉诸信任转移。借势明星、大V的脸或权威身份留人（包含明星同款妆容/穿搭等场景代入）。
    *   **核心边界（排他）**：依靠人设/身份光环留人。若为主讲人是普通素人无背书效应，不得打此标。

## 2. Effect 段（说服路径）
### 2.1 命名与考量（底层理论）
基于“消费心理学”的双轨说服模型进行重构：
1. **感性/现象层说服（E0-E4）**：向消费者展示“有多好”（眼见为实）。通过单纯的展示、强度递进的测评、横向对比以及五感放大，直接用现象说服。
2. **理性/逻辑层说服（E5-E7）**：向消费者拆解“为什么有效”（追根溯源）。化身老师、专家或见证者，通过教方法、拆原理、溯源头建立深层信任。

### 2.2 定义与核心边界
*   **E0 单点演示**
    *   **定义**：无门槛的普通状态、外观展示或常规使用步骤展示。
    *   **核心边界（排他）**：无教学门槛、无特定验证目的。若是常规涂抹、试穿，一律兜底至 E0。
*   **E1 效果测评**
    *   **定义**：有明确验证目标的常规实测验证（如喷水测眉笔防水、吸油纸测清爽度）。
    *   **核心边界（排他）**：常规物理测试，非极端高压环境。
*   **E2 暴力实测**
    *   **定义**：高压、破坏性、反常规的极端物理测试（如电钻测试、暴力摔打）。
    *   **核心边界（排他）**：常规的产品功能展示不属于极限测评，严禁越级滥用 E2。
*   **E3 对比/拉踩**
    *   **定义**：跟竞品、其他通用方案、旧方案或过去的自己进行明确优劣比较。
    *   **核心边界（排他）**：必须存在“本产品”与“竞品/方案”的明确优劣对比，或自身的强烈差异对比。
    *   **主体边界铁律**：凡是口播/字幕在讲“别人的、外面的、旧方案的、普通款的”成分、配料、添加剂、参数、安全性，并借此证明对方不好、旧方案不行、本品更值得替代，即使表面上提到了“成分/添加剂”，本质仍属于 `E3`，绝对不能判成 `E6`。
*   **E4 感官实证**
    *   **定义**：通过极致单感官输入触发大脑“通感”代偿，实现“以感代证”。
    *   **绝对边界（排他）**：极致的感官刺激必须能直接等价于产品的核心功效！如果单纯展示产品质地但不能直接推演核心功效，必须降级为 E0。长期起效型、隐性健康型品类，绝对禁止滥用 E4。
*   **E5 保姆级教程**
    *   **定义**：提供操作上的新方法、新认知或纠正旧方法，具备明确的指导和教学性质。
    *   **核心边界（排他）**：必须存在“认知增量”与“学习门槛”。单纯的按部就班产品使用流程展示（挤出-涂抹-冲洗），绝对不能打 E5，必须降级为 E0。
*   **E6 成分/参数科普**
    *   **定义**：提供客观的物质依据，包括化学成分解释、物理参数展示、专有技术名词，或官方的检疫/鉴定证书背书。
    *   **核心边界（排他）**：必须有确切的成分名词、数据参数、原理解释或官方证明支撑（不分高线成分党或下沉验真伪，只要诉诸客观参数/证书证据即属此类）。
    *   **主体边界铁律**：`E6` 只允许用于讲解“本产品自身”的成分、参数、证书、标准和原理。若讲的是外部方案、旧方案、他牌、普通款的成分/添加剂/参数，即使出现大量成分词，也不能打 `E6`，应优先考虑 `E3`。
*   **E7 产地溯源/工厂实录**
    *   **定义**：进入产品的原产地（如高端水源地/下沉农产地）或生产环境（如高科技无菌车间/市井手工作坊）进行实景展示。
    *   **核心边界（排他）**：以“源头一手实景、生产规模或透明度”为说服机制。普通室内展示不打此标。

## 3. CTA 段（收口机制）
### 3.1 命名与考量（底层理论）
基于“决策心理学”的双轨收口链路：
1. **转化/行为促动链路（C1-C3）**：强商业收口。诉诸理性算账（贪婪/价格）、风险规避（机制兜底）或行为捷径（直接指令），目标是立刻下单。
2. **心智/共鸣沉淀链路（C4-C5）**：内容弱收口。诉诸感性认同（身份闭环）与审美向往（效果定格），目标是强化心理一致性与长效心智。

### 3.2 定义与核心边界
*   **C1 利益/价格逼单**
    *   **定义**：诉诸贪婪与稀缺（FOMO）。用绝对低价或强烈的价格落差进行逼单。
    *   **核心边界（排他）**：必须聚焦在单一商品的价格维度或破价声明。如有额外实物赠送/特定机制，不打此类。
*   **C2 福利/保障机制**
    *   **定义**：诉诸安全感与风险厌恶。提供超预期价值（如买一送多、送正装量小样）或售后兜底（运费险、包退）。
    *   **核心边界（排他）**：必须出现“特定机制”或“售后兜底保障”，打消试错顾虑。纯降价不能越级打 C2。
*   **C3 指令行动**
    *   **定义**：诉诸行为捷径。下达极简的物理操作指令（如“左下角小黄车直接拍”）。
    *   **核心边界（排他）**：单纯的指引，不谈机制，不报具体价格。
*   **C4 人群/场景总结**
    *   **定义**：诉诸身份与逻辑闭环。在尾部再次圈定特定受众或情境呼应（如“干皮的姐妹冲”、“打工人必备”）。
    *   **核心边界（排他）**：侧重对号入座。若尾部叠加了强逼单机制，优先划入强商业收口（C1/C2）。
*   **C5 效果留白/情绪定格**
    *   **定义**：诉诸感官余温与审美向往。用极具冲击力的终效画面或治愈声音自然定格。
    *   **核心边界（排他）**：放弃叫卖，无指令无引导，让产品或画面本身说话。

## 4. 旧新版迁移映射铁律（供脚本与人工统一引用）
### 4.1 基本原则
1. **禁止只按编号硬映射。** 尤其是 Effect 段，旧版 `E1/E4/E5/E6` 与新版 `E1/E4/E5/E6` 存在语义换位，必须结合“内容机制”做语义判定。
2. **编号可复用，但语义不可偷换。** 例如旧版 `E4 教程/过程实拍` 不得因为编号相同而直接落到新版 `E4 感官实证`；必须改判为新版 `E5 保姆级教程`。
3. **迁移优先级**：先判定是否存在明确的教程 / 测评 / 感官 / 成分 / 溯源机制，再落到新版标签；无法证明时优先降级到 `E0 单点演示`，禁止拔高。
4. **所有迁移脚本必须带断言**：一旦产出标签不在最新版枚举内，或文本里残留废弃旧标签，脚本必须直接报错中断。

### 4.2 Hook / CTA 稳定别名映射
- `H1 痛点直击` → `H1 痛点/焦虑直击`
- `H2 反差结果前置` → `H3 反差结果前置`
- `H3 反常识悬念` → `H5 反常识与悬念`
- `H4 利益/价格前置` → `H2 利益/价格前置`
- `H6 场景代入` → `H6 场景/人群代入`
- `H7 即时操作展示` → `H4 即时操作展示`
- `H8 明星/权威同款` → `H7 明星/权威同款`
- `C1 利益逼单` → `C1 利益/价格逼单`
- `C2 指令行动` → `C3 指令行动`
- `C2 价格/福利诱导` → `C1 利益/价格逼单`
- `C3 人群/场景收口` → `C4 人群/场景总结`
- `C4 效果留白/情绪收尾` → `C5 效果留白/情绪定格`
- `C5 平台机制福利` → `C2 福利/保障机制`

### 4.3 Effect 语义强映射
- **旧版 `E1 保姆级教程`**：若核心是步骤、手法、纠错、认知增量，映射到新版 `E5 保姆级教程`；若本质是标准化验证，映射到新版 `E1 效果测评`；若实际在讲成分/参数，映射到新版 `E6 成分/参数科普`；若核心是强感官刺激，映射到新版 `E4 感官实证`；都不成立时降级到 `E0 单点演示`。
- **旧版 `E4 教程/过程实拍`**：强制映射到新版 `E5 保姆级教程`，绝不允许因为编号相同误落到新版 `E4 感官实证`。
- **旧版 `E4 成分/参数科普`**：映射到新版 `E6 成分/参数科普`；若内容并非成分原理而只是普通展示，则降级而非硬保留编号。
- **旧版 `E5 沉浸式场景演绎`**：不再保留为独立主标签；需根据真实机制改判为 `E0/E1/E4/E5/E6` 之一。
- **旧版 `E6 达人/真人生活化安利`**：不得直接等同新版 `E4 感官实证`；只有在存在强感官输入时才可落到 `E4`，否则需改判为 `E0/E1/E5/E6`。
- **旧版 `E2 极限测评`** → 新版 `E2 暴力实测`
- **旧版 `E3 对比拉踩`** → 新版 `E3 对比/拉踩`
- **旧版 `E7 产地溯源/工厂实录`** → 新版 `E7 产地溯源/工厂实录`

### 4.4 强制校验要求
- 任何 Python 交付脚本都必须同时校验：
  - HEC 主标签是否命中最新版枚举；
  - 辅标签是否命中同段枚举；
  - 输出文本中是否残留废弃旧标签；
  - 路径 / 任务等上游字典字段是否命中各自白名单。
- 若任一项不通过，必须 `raise AssertionError` 阻断，不允许继续写出报表。
